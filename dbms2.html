Types of databases

  1. Relational database

    These are databases which organize data into tables with rows and columns. Data points are related to each other through predefined relationships like primary and foreign keys. Example: MySQL.

  2. Object oriented databases

    These are databases which store data in the form of objects, similar to OOP. Used in CAD softwares

  3. NoSQL databases

    Also known as non Relational databases. They handle unstructured data and are highly scalable. Used to social media platforms. Example: 

                    
                    {
                        "id": "John Doe",
                        "email": "Johndoe@gmail.com",
